Votre navigateur ne supporte pas la 3D. Que faire ?

Google Chrome (Recommandé)

Google Chrome supporte nativement la 3D (webGL) depuis la version 18.
Pour savoir quelle version de chrome vous utilisez, allez dans Menu -> A propos de Google Chrome.
Il est conseillé d'utiliser la version la plus récente, pour des raisons de performances.

Si vous êtes sous linux et que webGL n'est pas activé sur Google Chrome, essayez la méthode suivante:

  • Tapez "chrome://flags" dans la barre d'url
  • Choisissez l'option "Ignorer la liste de rendu logiciel"
  • Relancer Google chrome (assurez vous d'avoir tué tous les processus)

Vous pouvez également lancer Google Chrome avec l'option "--ignore-gpu-blacklist", si webGL n'est toujours pas activé.

Mozilla Firefox

Mozilla Firefox supporte nativement webGL depuis la version 4. Il est recommandé d'utiliser la version la plus à jour.

Si la dernière version de Firefox ne vous permet pas d'utiliser webGL, essayez la manipulation suivante :

  • Allez dans les options de Firefox
  • Allez dans l'option "Avancé", puis dans "Général"
  • Cocher l'option “Utiliser l'accelération matérielle si disponible”
  • Relancez Firefox

Safari

Safari supporte la 3D par défaut depuis la version 8 (OS X Yosemite).
Pensez à mettre à niveau votre Mac, la mise à jour est gratuite !

Internet Explorer

Internet Explorer ne supporte pas encore suffisamment la 3D (webGL) pour faire fonctionner Cust, mais les ingénieurs de microsoft avancent à grand pas. Le support d'internet explorer sera intégré rapidement, en attendant, nous vous conseillons d'utiliser Google Chrome.

Opera

Opera supporte la 3D (webGL) depuis la version 15, mais le manque du clic droit peut s'avérer génant sur certains outils.

Si la 3D ne fonctionne pas sur Opera, vous pouvez essayer la manipulation suivante:

  • Mettre la valeur "opera:config#Enable%20Hardware%20Acceleration" à 1
  • Mettre la valeur "opera:config#Enable%20WebGL" à 1
  • Relancez Opera
